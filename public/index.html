Timestamp microservice - API

You can use API by passing string parameter either as a unix timpestamp or as a natural language date.

The API will return both the Unix timestamp and the date in the normal Month, Day, Year form.

If you pass parameter as a natural language date, the month must be written with letters, not with numbers.

If you pass an incorrect string parameter, the API will return null values for the JSON object properties.

Usage

  • natural language date: https://timestamp-ms.herokuapp.com/December%2015,%202015
  • unix timestamp: https://timestamp-ms.herokuapp.com/1450137600

Output

  • correct request: { "unix": 1450137600, "natural": "December 15, 2015" }
  • incorrect request: { "unix": null, "natural": null }
